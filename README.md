# yt-dlp-gui

A video downloading tool based on yt-dlp.  
It provides a graphical user interface, allowing users to use yt-dlp without manual environment configuration.

If you need to download files in other formats, you can customize the command accordingly.

---

## How to Obtain Cookies

Using the Edge browser as an example:

1. Install the extension **Cookie-Editor**
2. Open Cookie-Editor
3. Click the **Export** button (bottom right)
4. Select the **Netscape format**
